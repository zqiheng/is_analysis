# 接口：setPassword  [返回](../README.md)
用例： [修改密码](../用例/修改密码.md)

- 功能：用户修改密码。

- 权限：学生、老师、管理员，修改自己的密码，必须先登录。

- API请求地址： 接口基本地址/v1/api/setPassword?id=? and password=?

- 请求方式 ：
    POST

- 请求实例：

        {
            "id":1234,
            "password":12345678
        }

- 请求参数说明:

  |参数名称|说明|
  |:-:|:--|
  |id|用户的ID号。对应表Users.user_id的值|
  |password|用户的密码。不能为空。|

- 返回实例：

        {
            "status": true,
            "info": null,

        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回(修改成功)，false表示有错误|
  |info|返回结果说明信息|
